# Snsu-OTJ

Multilingual LGU chatbot with text-to-speech support (English and Filipino).

## What You Need to Install

- **Python** 3.8 or newer
- **pip** (usually comes with Python)

### Dependencies

Install all required packages:

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install flask>=3.0.0 edge_tts
```

## Running the App

```bash
python app.py
```

Then open [http://127.0.0.1:5000](http://127.0.0.1:5000) in your browser.
